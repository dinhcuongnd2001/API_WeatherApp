Media_Query

dùng các câu truy vấn để xác định kích thước hiển thị

Cấu trúc: @media not|only mediatype and (mediafeature and|or|not mediafeature){ CSS-Code }

Keyword

  1. not
  2. only
  3. and
  4. or

Mediatypes: Những trường hợp muốn CSS

  1. print
  2. screen
  3. speech
  4. all-default

MediaFeatures: là những value lựa chọn ra những trường hợp cụ thể muốn CSS

  1. print
  2. screen
  3. speech
  4. all-default
